# Analytics

This folder contains the Python analysis workflow used to prepare campaign performance and attribution data for Tableau dashboarding.

## Main Notebook

`TV_Advertising_Impact_Analysis.ipynb`

This notebook covers:

- data loading and inspection
- network name standardization
- missing and invalid value handling
- KPI calculation for CPM, CPV, CR, and CPA
- monthly and network-level aggregation
- traffic-response attribution preparation
- Tableau-ready dataset export

The notebook supports the final dashboard and executive recommendations in this portfolio case study.

## Note

Original source data is not included in this public repository. The workflow is documented using sanitized sample data and derived analysis logic.
